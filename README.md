# FloppySanta
Park Plaza Pals' final project :) 

Testing Instructions: 
1. Open terminal and run: "chmod +x install.sh"
2. Run: "./install.sh"
3. Open a seperate terminal and run: "chmod +x run.sh"
4. Run: "./run.sh"
5. Open localhost:8000 to play the game!
